Yêu cầu tối thiểu của hệ thống:

  • CPU: Pentium 4, 1.8 Mhz hoặc cao hơn
  • RAM: tối thiểu 512 MB, đề nghị: 1 GB
  • Hệ điều hành: Windows XP, Windows Vista, Windows 7, Windows 8
  • Ghi chú: Nếu cài đặt không thành công, bạn hãy thử giảm độ bảo mật hoặc tắt hẳn chương trình diệt virus, rồi thực hiện lại từ đầu quá trình cài đặt.
